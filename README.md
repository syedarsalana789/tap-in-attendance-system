# Tap In: No More Proxy, Just One Tap

Tap In is a smart attendance management system designed for faculty and staff using biometric fingerprint and facial recognition.

## Overview
The system records in/out timings, stores data in a centralized database, and allows monitoring of attendance, leaves, and working hours.

## Problem Statement
Traditional systems are manual and error-prone, leading to proxy attendance and inefficient management of shifts and leaves.

## Methodology
Attendance is marked using biometric devices. Data is sent to the server, processed, and stored. The system automatically detects late arrivals and early departures.

## System Roles

### User Panel
- View attendance records  
- Apply for leaves  
- Check late and early status  

### Admin Panel
- Manage all staff  
- Approve/reject leaves  
- Manage shifts and holidays  

### Department Panel
- Manage department users  
- View attendance  
- Approve leave requests  

## Features
- Biometric Attendance (Fingerprint & Face)
- Automatic Late/Early Detection
- Leave Management System
- Role-based Dashboards
- Real-time Tracking

## Scope
Suitable for universities and organizations to manage attendance and staff efficiently.
